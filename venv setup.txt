2. 가상환경에서 공용공간 모듈 활용(공용 모듈버전 포함)         
터미널에서
    python -m venv mvenv --system-site-packages            
    -이하-가상환경 활성화까지 동일
터미널에서 pip list 하면 공용공간에 목록
    pip list --local 하면 가상환경 목록 


3. 가상환경 복제 방법
pip freeze > requirements.txt 
로 파일을 만들고, 그것만 git에 add 해서 올리고,
복제할 때에는, 
pip install -r requirements.txt
로, 동일한 venv 를 생성합니다.
