1. pip install django
(python) django-admin startproject 생성할폴더명 .(url 설정 서버라 생각하면 됨)
python manage.py startapp 생성할app폴더명(구동 views 포인트 )
python manage.py migrate(데이터베이스구축)
python manage.py createsuperuser(관리자 계정만들기)
python manage.py migrate(저장)
python manage.py runserver(서버 실행)


2. 서버 setting파일
    INSTALLED_APPS 부분에 app폴더내 
    app 함수(ex/blog.apps.BlogConfig)를 써주면 됨.

    TEMPLATES 부분에 import os 하고 
        'DIRS': [os.path.join(BASE_DIR, 'templates')],

    STATIC_URL = '/static/' 아래에 추가
    STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]


3. 서버 urls파일
    from django.urls import path
    from 생성앱명.views import Main
    urlpatterns = [
        path('', Main.as_view())
    ]


4. 앱 views파일(pip install djangorestframework 설치해야 됨)
    from django.shortcuts import render
    from rest_framework.views import APIView
    class Main(APIView):
    def get(self, request):
        return render(request, '페이지로 쓸.html')


5. 앱 templates폴더를 생성해서 거기 '페이지로 쓸.html' 파일로 작업.
